# Anforderungsanalyse #
## **Grocery Mate Webshop** ##
## *Fragen* ##
### *Registrierung* ###
1. Ist eine Registrierung notwendig oder kann man auch als Gast bestellen?
- Eine Registrierung ist notwendig
2. Gibt es ein Mindestalter für die Registrierung? 
- Nein 
3. Gibt es Einschränkungen für minderjährige Nutzer?
- Um Alkoholische Produkte zu kaufen, muss man mindestens 18 Jahre alt sein.
### *Bestellabschluss* ###
1. Welche Versandarten gibt es?
- Im Bestellformular ist keine Auswahl für Versandarten vorhanden. Daher ist nicht ersichtlich, welche Versandarten angeboten werden. 
2. Sind Bestellungen aus dem Ausland möglich?
- Im Bestellformular gibt es keine Auswahl für das Versandland.
3. Welche Zahlfunktionen werden unterstützt?
- Im unteren Bereich werden 6 Zahlungsmethoden angeboten (Visa, Google Pay, Apple Pay, Mastercard, Amex, ?)
4. Wie wählt der Benutzer die gewünschte Zahlungsmethode aus?
- eine Auswahlmöglichkeit für die verschiedenen Zahlungsmethoden ist nicht erkennbar. Es steht jedeglich ein Kreditkartenformular zur Verfügung. 

### *Produkte in den Warenkorb legen* ###
1. Gibt es eine Meldung wenn ein Produkt ausverkauft ist?
- Im Test konnten keine als ausverkauft markierten Produkte gefunden werden. 
2. Können ausverkaufte Produkte trotzdem in den Warenkorb gelegt werden?
- Nicht testbar. 
3. Gibt es eine maximale Bestellmenge 
- Im Test konnte keine Maximale Bestellmenge festgestellt werden.  
### *Produktsuche* ###
1. Werden Suchanfragen bei unvollständiger Eingabe ergänzt?
- Ja. Unter dem Suchfeld erscheint eine Auswahl passender Artikel die angeboten werden.
2. Gibt es Alternativvorschläge wenn kein passendes Produkt gefunden wird?
- Nein. 
3. Gibt es eine Meldung wenn die Suche kein Ergebnis liefert?
- Nein.
### *Produkte zu Favoriten hinzufügen* ###
1. Wo werden die Favoriten angezeigt?
- Oben in der blauen Navigationsleiste. Desweiteren gibt es ein Herzsymbol (ganz oben rechts) zum Öffnen der Favoriten. 
2. Woran erkennt man, dass ein Produkt favorisiert wurde?
- Unten den Produkten gibt es 3 Navigationselemente. (Auswahl der Menge, Hinzufügen zum Warenkorb und ein Herzsymbol)Wenn das Herz blau ist, wurde der Artikel favorisiert.
3. Bleiben Favoriten gespeichert wenn man sich ausloggt?
- Ja. 
### *Produktsuche mit Sortierfunktion* ###
1. Nach welchen Kriterien kann sortiert werden?
- Empfohlene Produkte, Alphabetisch nach Name (von A-Z), Preis (aufsteigend). 
2. Gibt es eine maximale Anzahl bei den Suchergebnissen?
- Pro Seite werden maximal 12 Produkte angezeigt. 
3. Können mehrere Filterkriterien ausgewählt werden um die Suche einzuschränken?
- Nein.
